Bug List


Known problems

  • The JNI/RNI java interface has a bug in the palette setting code. The palette is always set to a blue gradient until the code is fixed. Hold tight java users (or use truecolor :)

Little annoying things...

  • Window positions may get messed up a bit when switching in and out of fullscreen mode with ALT-TAB.
  • Pressing the ALT or F10 keys pauses the program until another key is pressed because these keys are win32 system keys.
